# Button

A versatile button component that supports multiple intents (primary, secondary, danger, ghost, outline), sizes, and states. Uses design tokens from `design-tokens.css` for consistent styling.

## Usage

```tsx
import { Button } from './components'

function MyComponent() {
  return (
    <Button 
      label="Click me" 
      intent="primary" 
      size="md" 
      onClick={() => console.log('clicked')}
    />
  )
}
```

## Variants

### Primary

Primary buttons are used for main actions or calls to action. They have the highest visual prominence.

### Secondary

Secondary buttons are used for secondary actions that complement primary actions.

### Danger

Danger buttons are used for destructive or dangerous actions.

### Ghost

Ghost buttons have a transparent background and are used for low-priority actions.

## Design Tokens

The Button component uses the following design tokens:

- `--color-primary` - Primary button background
- `--color-primary-hover` - Primary button hover state
- `--color-primary-active` - Primary button active state
- `--color-error` - Danger button background
- `--color-error-light` - Danger button hover state
- `--color-error-dark` - Danger button active state
- `--color-bg-secondary` - Secondary button background
- `--color-bg-tertiary` - Secondary button hover state
- `--color-text-primary` - Text color
- `--color-text-inverse` - Inverse text color
- `--radius-theme` - Border radius
- `--font-sans` - Font family
