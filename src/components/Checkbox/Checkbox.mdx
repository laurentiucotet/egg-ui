# Checkbox

A checkbox component that allows users to select one or more options. Supports checked, unchecked, and indeterminate states. Uses design tokens from `design-tokens.css` for consistent styling.

## Usage

```tsx
import { Checkbox } from './components'

function MyComponent() {
  return (
    <Checkbox 
      label="Accept terms" 
      onChange={(checked) => console.log(checked)}
    />
  )
}
```

## Design Tokens

The Checkbox component uses the following design tokens:

- `--color-primary` - Checkbox background when checked
- `--color-primary-hover` - Checkbox hover state when checked
- `--color-border` - Default border color
- `--color-bg-primary` - Background color
- `--color-text-primary` - Label text color
- `--radius-sm` - Border radius
- `--transition-fast` - Transition duration
- `--font-sans` - Font family
- `--line-height-normal` - Label line height
