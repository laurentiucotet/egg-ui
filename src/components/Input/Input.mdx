# Input

A versatile input component that supports multiple input types, variants, sizes, and includes features like labels, icons, and password strength indicator. Uses design tokens for consistent styling.

## Usage

```tsx
import { Input } from './components'

function MyComponent() {
  return (
    <Input
      label="Email"
      showLabel
      type="email"
      placeholder="Enter your email"
      icon="Mail"
      required
    />
  )
}
```

## Password Strength

The password input includes a strength indicator that shows:

- Weak: Less than 8 characters
- Medium: At least 8 characters with mixed case and numbers
- Strong: At least 8 characters with uppercase, lowercase, numbers, and special characters

## Design Tokens

The Input component uses the following design tokens:

- `--color-bg-primary` - Input background
- `--color-bg-inverse` - Dark mode input background
- `--color-text-primary` - Text color
- `--color-text-inverse` - Dark mode text color
- `--color-text-muted` - Placeholder text color
- `--color-border` - Border color
- `--color-primary` - Focus border color
- `--color-primary-hover` - Dark mode focus border
- `--color-error` - Error border color
- `--color-success` - Success border color
- `--radius-theme` - Border radius
- `--font-sans` - Font family
