# LoginForm

A login form component with email and password fields, wrapped in a card component. The form includes built-in validation that prevents submission when required fields are empty.

## Usage

<Source language="tsx" code={`import { LoginForm } from './LoginForm'

// Basic usage
<LoginForm onSubmit={(data) => console.log(data)} />

// With custom styling
<LoginForm 
  onSubmit={(data) => console.log(data)} 
  className="w-full max-w-md mx-auto"
/>`} />

## Props

<Controls />

## LoginFormData

The data shape returned when the form is successfully submitted.

| Property | Type | Description |
|----------|------|-------------|
| email | `string` | The user's email address |
| password | `string` | The user's password |

## Features

### Form Validation
- Both email and password fields are required
- Form submission is prevented if either field is empty
- Visual validation through HTML5 required attribute

### Built-in Components
The LoginForm composes several existing components:
- **[Card](/?path=/docs/components-card--docs)** - Main container with card styling
- **[CardHeader](/?path=/docs/components-card--docs)** - Header with "Login" title and description
- **[CardContent](/?path=/docs/components-card--docs)** - Form content wrapper
- **[Input](/?path=/docs/components-input--docs)** - Email and password input fields
- **[Button](/?path=/docs/components-button--docs)** - Submit button
